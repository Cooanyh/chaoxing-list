## 2026-03-12 进展（v2.1.0）

- **布局修复**:
    - 课程进度卡片（`.progress-card`）在宽屏下左右两列等宽问题已修复。
    - 给 `.progress-item` 添加 `min-width: 0` 和 `overflow: hidden`，防止内容撑破 grid 列，消除横向滚动条。
    - 移动端（`max-width: 768px`）下，进度卡片从跨双列改为单列堆叠（`grid-column: span 1`）。

- **忽略功能（新增）**:
    - 所有板块（待办任务、作业、考试、课程任务、课程进度）的每条课程信息均添加了 🚫 忽略按钮（hover 时显示）。
    - 点击忽略按钮弹出二次确认对话框，列出将被忽略的项目名称，支持点击遮罩取消。
    - 支持多选忽略：在"查看全部"详情页顶部点击"多选"切换多选模式，每项显示复选框，底部出现"忽略所选(N)"批量操作栏。
    - 忽略状态通过 `localStorage` 持久化存储（键名 `chaoxing_ignored_items`），刷新后保持。
    - 各板块使用 Vue `computed` 过滤已忽略项，忽略后立即从列表中消失。
    - 在"查看全部"详情页顶部新增"🚫 已忽略(N)"按钮，点击进入已忽略内容面板：
        - 显示所有已忽略项（标题、课程、忽略日期）。
        - 每项有"撤销忽略"按钮，支持多选后批量恢复。

- **版本号**:
    - 版本号更新为 **2.1.0**。
    - 完成代码提交与推送。

## 2026-03-13 进展（v2.2.0）

- **增强版课程进度悬浮窗（新增）**:
    - 在课程进度卡片中，当鼠标移动到某个课程时，显示增强版悬浮窗。
    - 悬浮窗展示该课程的所有学习活动，包括：
        - 📝 课程作业（显示标题和状态：进行中/已完成/未开始）
        - 📋 在线考试（显示标题和状态）
        - 📖 章节测验（显示标题和状态）
        - 💬 互动测验（包括讨论、抢答、评分等，显示标题和状态）
    - **智能数据加载**：
        - 鼠标悬停时自动通过 API 加载活动数据
        - 数据缓存机制，避免重复请求
        - 每个分类最多显示3条活动
        - 过滤掉签到、投票、直播等非学习类活动
    - **悬浮窗交互优化**：
        - z-index 提升至 10000，确保不被其他元素遮挡
        - 悬浮窗宽度从 240px 增加到 280px，提供更多空间
        - 添加最大高度和滚动条，防止内容过多溢出
        - 移动端自适应：小屏幕下悬浮窗居中显示在课程卡片下方
    - **视觉优化**：
        - 活动列表使用彩色边框区分状态：蓝色（进行中）、绿色（已完成）、灰色（未开始）
        - 每个活动显示状态标签（进行中/已完成/未开始）
        - 分类标题使用图标标识，易于识别
        - 加载状态友好提示："正在加载活动..."

- **技术实现细节**:
    - 新增 Vue 响应式数据：`courseActivitiesCache` 用于缓存活动数据，`hoveredCourseKey` 追踪当前悬停的课程
    - 新增函数：`fetchCourseActivitiesOnHover()` 用于悬停时获取数据，`handleProgressItemHover()` 和 `handleProgressItemLeave()` 处理悬停事件
    - 复用已有的 `fetchCourseActivities()` 函数获取课程活动列表
    - 增强 CSS 样式：新增 `.tooltip-activities-section`、`.activity-group-title`、`.activity-item` 等样式类
    - 响应式设计：添加 `!important` 规则确保移动端布局正确
